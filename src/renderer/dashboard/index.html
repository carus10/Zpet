How It Works

IDE Watcher

Monitors the Antigravity IDE conversations directory for database file changes. When the IDE is active, the pet shows "working." When it pauses, it transitions to "waiting" then "idle."

Point to the folder containing .db files. Default: ~/.gemini/antigravity-ide/conversations

CLI Watcher

Monitors the Antigravity CLI brain directory for log file changes. Automatically finds the most recently active session.

Point to the brain directory containing session subdirectories. Default: ~/.gemini/antigravity-cli/brain

Watch Mode

"IDE Only" monitors just the IDE. "CLI Only" monitors just the CLI. "Both" monitors both simultaneously.
